# 磁盘性能测试工具

一个基于WPF开发的Windows桌面应用程序，用于测试磁盘性能。

## 功能特性

1. **可配置的测试参数**
   - 测试路径选择
   - 测试类型（顺序读取/写入、随机读取/写入、混合测试）
   - 文件大小、块大小、线程数、队列深度、持续时间等参数
   - 支持使用Windows SAT (WinSAT) 或自定义测试

2. **实时命令预览**
   - 修改配置参数后自动显示待执行的命令预览

3. **测试结果展示**
   - 读取/写入速度 (MB/s)
   - 读取/写入 IOPS
   - 读取/写入延迟 (ms)
   - 测试状态和进度信息

4. **测试历史记录**
   - 自动保存所有测试结果
   - 查看历史测试记录
   - 查看详细测试结果

## 系统要求

- Windows 10/11
- .NET 8.0 Runtime

## 构建说明

1. 确保已安装 .NET 8.0 SDK
2. 在项目目录下运行：
   ```bash
   dotnet build
   ```
3. 运行应用：
   ```bash
   dotnet run
   ```

## 使用方法

1. **配置测试参数**
   - 选择测试路径（点击"浏览..."按钮）
   - 选择测试类型
   - 设置文件大小、块大小、线程数、队列深度、持续时间等参数
   - 可选择使用WinSAT或自定义测试

2. **查看命令预览**
   - 修改任何配置参数后，左侧"待执行命令"区域会自动更新显示将要执行的命令

3. **开始测试**
   - 点击"开始测试"按钮
   - 测试过程中可以查看实时进度
   - 可以点击"停止测试"按钮取消测试

4. **查看结果**
   - 测试完成后，右侧"测试结果"区域会显示关键性能指标
   - 点击"查看历史"按钮可以查看所有历史测试记录
   - 在历史记录窗口中可以查看每条记录的详细信息

5. **重置配置**
   - 点击"重置配置"按钮可以恢复所有参数到默认值

## 默认配置

- 测试路径: `C:\`
- 文件大小: 100 MB
- 块大小: 1024 KB
- 线程数: 1
- 队列深度: 1
- 持续时间: 30 秒
- 测试类型: 顺序读取
- 使用WinSAT: 否

## 注意事项

- 测试会在指定路径创建临时文件，测试完成后会自动删除
- 确保有足够的磁盘空间进行测试
- 测试过程中可能会影响系统性能，建议关闭其他占用磁盘的程序
- 使用WinSAT需要管理员权限

## 技术栈

- WPF (.NET 8.0)
- Newtonsoft.Json (用于历史记录序列化)
